# TaskTracker CLI

Консольное приложение для управления задачами, написанное на Go.

## Функциональность

### Основные возможности:
- Добавление задач с описанием
- Обновление существующих задач
- Удаление задач
- Просмотр всех задач
- Изменение статуса задач
- Фильтрация задач по статусу

## Установка

```bash
# Клонирование репозитория
git clone https://github.com/YourUsername/TaskTracker_CLI.git
cd TaskTracker_CLI

# Сборка приложения
go build -o task-tracker main.go
